# SEO Roadmap (2 semanas) - Jahatelo

## Objetivo
Mejorar visibilidad orgánica, CTR y posicionamiento local para búsquedas relacionadas con moteles en Paraguay.

## Semana 1 (base técnica + on-page)

### 1) SEO técnico (prioridad alta)
- [ ] Validar `canonical` en páginas indexables.
- [ ] Asegurar `noindex` en staging/preview.
- [ ] Revisar `robots.txt` y `sitemap.xml` con `lastmod` real.
- [ ] Auditar Core Web Vitals (LCP, INP, CLS) en home y detalle de motel.

### 2) Metadata por plantilla (prioridad alta)
- [ ] Home: title + description comercial optimizada.
- [ ] Search: metadata alineada a intención de búsqueda.
- [ ] Ciudades: metadata específica por ciudad.
- [ ] Detalle motel: title/description únicos con ciudad y atributos clave.

### 3) Estructura SEO on-page (prioridad alta)
- [ ] Un solo `H1` por página.
- [ ] Jerarquía `H2/H3` consistente.
- [ ] Mejorar textos intro en home/ciudades/search con keywords naturales.
- [ ] Revisar links internos entre home, ciudades, mapa y moteles.

## Semana 2 (SEO local + contenido + schema)

### 4) Schema markup (prioridad alta)
- [ ] `Organization` global.
- [ ] `WebSite` con `SearchAction`.
- [ ] `BreadcrumbList` en páginas de listado y detalle.
- [ ] `LodgingBusiness` por motel con datos completos.
- [ ] `FAQPage` en páginas estratégicas.

### 5) SEO local (prioridad alta)
- [ ] Crear/optimizar landings por ciudad con contenido único.
- [ ] Agregar texto útil por zona/barrio (evitar thin pages).
- [ ] Crear secciones “Moteles con jacuzzi”, “por hora”, etc. por ciudad.

### 6) Tracking y mejora continua (prioridad alta)
- [ ] Search Console: revisar queries, CTR y páginas con muchas impresiones.
- [ ] GA4: eventos clave en páginas SEO.
- [ ] Ajustar titles/descriptions según performance real cada 2 semanas.

## Backlog (después de 2 semanas)
- [ ] Plan de backlinks locales (directorios, prensa, aliados).
- [ ] Google Business Profile de marca y activos de reputación.
- [ ] Programación editorial mensual de contenido SEO.

## KPIs de seguimiento
- Impresiones orgánicas.
- CTR orgánico.
- Posición media por keyword objetivo.
- Tráfico orgánico por landing.
- Conversiones (click WhatsApp, click llamada, visitas a detalle de motel).

## Notas
- La meta tag `keywords` no mueve posicionamiento en Google moderno.
- Priorizar: contenido útil + velocidad + schema + arquitectura de enlaces internos.
