# BRAND KIT - PRESENTACIONES JAHATELO
## Guia oficial para diseno de propuestas comerciales y PPT

Ultima actualizacion: 2026-04-13

---

## 1) Identidad visual (resumen)

Jahatelo combina:
1. Base morada/lila (tecnologia + confianza)
2. Acentos pink/fucsia (energia comercial)
3. Blanco y grises neutros (claridad y lectura)

Estilo recomendado para ventas:
1. Premium, limpio, con contraste alto
2. Pocas frases por slide, impacto visual
3. Mucho aire (espacios), nada recargado

---

## 2) Paleta oficial (hex)

### Colores principales de marca
1. `#822DE2` - Morado principal (brand primary)
2. `#F56565` - Pink coral (acento CTA)
3. `#D53F8C` - Pink oscuro/fucsia (acento gradiente)
4. `#FFFFFF` - Blanco base
5. `#424A54` - Gris texto institucional

### Gradiente de marca oficial
1. `linear-gradient(135deg, #F56565 0%, #D53F8C 50%, #822DE2 100%)`

### Acentos comerciales por plan
1. Basico: `#822DE2`
2. Gold: `#F6C90E`
3. Diamond: `#00D4FF`

### Fondos oscuros usados en la web
1. `#0F0F1A`
2. `#161624`
3. `#1A1530`

Uso recomendado:
1. Slides de impacto/portada: fondo oscuro + texto blanco
2. Slides de informacion/tabla: fondo claro + texto oscuro

---

## 3) Tipografias (letras)

### Tipografia principal oficial
1. `Lato` (web usa Lato 400/700)

### Fallbacks permitidos
1. `Inter` (cuerpo)
2. `Poppins` (solo titulos, si se busca look mas comercial)

### Jerarquia recomendada para PPT
1. Titulo de slide: 40-48 pt, bold
2. Subtitulo: 18-24 pt, semibold/regular
3. Cuerpo: 18-24 pt
4. Nota legal/footer: 10-12 pt

Regla:
1. Nunca usar mas de 2 familias tipograficas en una misma presentacion.

---

## 4) Logos y assets oficiales

### Archivos base (usar estos)
1. Isotipo: `web/jahatelo-web/public/logo-icon.png`
2. Logotipo texto gradiente: `web/jahatelo-web/public/logo-text-gradient.png`
3. Composicion completa: `app/jahatelo-app/assets/logo-full.png`
4. Master alternativo: `web/jahatelo-web/public/logo-master.png`

### Reglas de uso
1. Fondos claros: preferir `logo-text-gradient.png`
2. Fondos oscuros: usar logo en version que mantenga contraste (blanco o gradiente con halo sutil)
3. No deformar proporciones (siempre escalar con lock de aspecto)
4. No agregar bordes, sombras duras ni contenedores innecesarios al logo

### Area de seguridad (clear space)
1. Dejar minimo 1x la altura del icono alrededor del logo
2. Nunca pegar logo a bordes o texto

---

## 5) Estilo visual para presentaciones comerciales

### Composicion
1. Maximo 1 idea principal por slide
2. Maximo 4 bullets por bloque
3. Evitar parrafos largos
4. Usar tarjetas/cards para planes y beneficios

### Componentes recomendados
1. Cards blancas con borde suave `#E9E3F7`
2. CTA con color principal `#822DE2`
3. Etiquetas de plan: Basico morado, Gold dorado, Diamond cyan
4. Lineas/ornamentos con gradiente de marca en separadores

### Iconografia
1. Estilo outline o duotono simple
2. Mismo grosor en toda la presentacion
3. Evitar mezclar estilos de iconos distintos

## 8) Prompt breve para diseniador (copy/paste)

Disenar una presentacion comercial premium para Jahatelo usando este sistema visual:
- Marca principal: morado `#822DE2`
- Gradiente oficial: `#F56565 -> #D53F8C -> #822DE2`
- Acentos de planes: Gold `#F6C90E`, Diamond `#00D4FF`
- Tipografia principal: Lato (titulos bold, cuerpo regular)
- Estilo: limpio, moderno, alto contraste, pocas palabras por slide
- Mantener consistencia de iconos y espaciado generoso
- Incluir logo oficial sin deformar ni encerrar en cajas
- Preparar version 16:9 para presentacion comercial B2B
